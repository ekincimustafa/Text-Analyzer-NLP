# Natural Language Processing: Text Statistics Analyzer

This repository contains a Python-based text analysis tool designed to perform basic Natural Language Processing (NLP) tasks. The program reads a text file, processes its contents by stripping punctuations, and generates detailed lexical statistics, including word frequencies, sentence counts, and character analysis.

This project was developed as part of the BBM103 course assignments, focusing on string manipulation, file I/O operations, and statistical data processing.

## 🚀 Features

- **Lexical Analysis:** Calculates the total number of words, sentences, and characters (both with and without spaces).
- **Word Frequency Distribution:** Determines the exact frequency of each word in the text, formatting the output to four decimal places.
- **Extremes Identification:** Automatically identifies and extracts the shortest and longest words in the document.
- **Punctuation Handling:** Safely strips various punctuation marks (`.,!?()'[]{}:;"`) to ensure accurate word counting and analysis.
- **Command-Line Interface (CLI):** Processes dynamic input and output file paths directly via command-line arguments.

## 🛠️ Tech Stack
- **Language:** Python 3.9+
- **Libraries:** `sys` (for CLI arguments), `locale` (for standardizing output formatting)

## 📦 Input Format
The script processes standard `.txt` files containing any English text (e.g., articles, essays, logs).

## ⚙️ How to Run

1. Clone the repository:
   ```bash
   git clone [https://github.com/ekincimustafa/Text-Analyzer-NLP.git](https://github.com/ekincimustafa/Text-Analyzer-NLP.git)
   ```
2. Navigate to the project directory:
   ```bash
   cd Text-Analyzer-NLP
   ```
3. Run the Python script via terminal, providing the input text file and the desired output text file as arguments:
   ```bash
   python text_analyzer.py i1.txt o11.txt
   ```
4. The script will analyze the text inside `i1.txt` and generate the statistical report in `o11.txt`.
